Tecnologia Game Shift: Aumente a potência quando os jogos ficarem difíceis precionando um simples botão. O Game Shift é ativado precionando FN + Tecla Game Shift e ativa o modo dinâmico no Alienware Command Center, maximizando a velocidade dos ventiladores para manter o sistema resfriado enquanto os processos trabalham mais.

Som claro: Os auto-falantes duais com Nahimic 3D audio for Gamers permitem que você ouça todos os planos de ataques com nitidez incrível.

Iluminação incrível: Jogue com nitidez em configurações de pouco luminosidade com o teclado retroiluminado RGB opcional de quatro zonas com WASD controlado pelo Alienware Command Center.

Energia que dura: Com a nova bateria de 56 Wh (68 Wh no RTX 3060) que carrega o sistema rapidamente, seus jogos não ficam limitados a uma tomada.

Alienware Command Center na Série G da Dell

Torne-se imbatível: Criamos um software inovador especificamente projetado para melhorar a experiência gamer, ajustar todas as configurações de games e sistemas, solucionar problemas e gerenciar o tempo.

Calibre os controles: Agora você tem controle sobre seus ambientes de hardware e software. Além de adaptar o hardware de acordo com cada game, você pode personalizar o espaço e a configuração para deixar seu software exatamente como deseja, sem interferir na jogabilidade.

Biblioteca vinculada: A interface fácil de usar vincula todas as configurações do sistema a uma biblioteca de jogos, simplificando o acesso às informações de todos eles. Independente da fonte, o Command Center oferece acesso fácil aos games e permite ajustar a configuração de cada um deles.
